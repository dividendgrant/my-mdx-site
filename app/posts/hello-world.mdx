Welcome to my site! This is the first post, written entirely in **MDX**.

## What is MDX?

MDX lets you use JSX directly inside Markdown files. That means you can import
and render React components alongside your prose.

## Why it's useful

- Write content in familiar Markdown syntax
- Embed interactive components where you need them
- Keep everything in one file

Next.js gives you everything you need to build fast, production-ready React apps.

## App Router

The App Router uses React Server Components by default. Pages are co-located
with their layouts, loading states, and error boundaries.

## File conventions

| File | Purpose |
| --- | --- |
| `page.tsx` | Renders the route UI |
| `layout.tsx` | Shared UI across routes |
| `loading.tsx` | Streaming suspense fallback |
| `error.tsx` | Error boundary |
